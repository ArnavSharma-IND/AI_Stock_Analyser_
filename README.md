# 🚀 AI Stock Analyzer Web App

A modern **AI-powered Stock Analysis Dashboard** with a **crypto trading UI** design.
Built using **HTML, CSS, and JavaScript**, this project simulates real-time stock data, performs technical analysis, and generates intelligent investment insights.

---

## 🌟 Features

### 📊 Real-Time Stock Simulation

* Dynamic stock price generation
* Live updates with realistic trends
* Key metrics: price, volume, change %

### 📈 Advanced Charting

* Interactive price charts
* Moving averages visualization
* Smooth animations

### 🧠 AI Prediction Engine

* BUY / SELL / HOLD signals
* Confidence percentage
* Predicted price output
* Explanation of reasoning (RSI, trend, momentum)

### ⚙️ Technical Indicators

* SMA (Simple Moving Average)
* EMA (Exponential Moving Average)
* RSI (Relative Strength Index)
* MACD (Moving Average Convergence Divergence)

### 💰 Investment Calculator

* Input investment amount (₹)
* Estimated returns
* Risk level analysis
* Growth projection

### 🔥 Market Dashboard

* Top gainers & losers (mock data)
* Market sentiment meter (Bullish/Bearish)

### 📁 Portfolio Tracker

* Add/remove stocks
* Track profit/loss
* Data stored using localStorage

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Charts:** Chart.js / Canvas
* **Storage:** LocalStorage
* **Logic:** Custom AI simulation algorithms

---

## 🧠 How It Works

* Generates realistic stock data using simulation
* Applies technical indicators (RSI, MA, MACD)
* Uses rule-based AI logic for predictions
* Displays insights in a trading dashboard UI
